## Continue learning Miranda

This extension ships a full language tutorial based on David Turner's
[*An Overview of Miranda*](https://www.cs.kent.ac.uk/people/staff/dat/miranda/Overview.html).

### Topics covered

1. Basic ideas — scripts, lists, tuples
2. Guarded equations and `where` blocks
3. Pattern matching
4. Higher-order functions and currying
5. List comprehensions
6. Lazy evaluation and infinite lists
7. Polymorphic typing with `::`
8. User-defined types with `::=`
9. Type synonyms with `==`
10. Abstract data types with `abstype`
11. Separate compilation with `%include`

### Example files

Open `examples/01-basic-ideas.m` through `examples/10-abstract-data-types.m`
from the extension folder, or read `docs/walkthrough.md` for the guided tour.
